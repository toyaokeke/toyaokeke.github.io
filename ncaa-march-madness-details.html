NCAA MArch Madness Database

Project Information

  • Category: Web Design
  • Number of Teammates: 2
  • Project URL: Click here to open the repository.

I joined two other students in building web application that contains March Madness tournaments from 2015-2019. The backend is an SQLite database which responds to a frontend built using PHP.
Due to the COVID-19 pandemic, we decided to use the data from the Kaggle competition to build the database, and later plan to build machine learning pipeline RDDs with Spark to make predictions for next year's tournament.
