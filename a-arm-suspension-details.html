Using Design Studies to create an A-arm

Project Information

  • Category: Numerical Simulation
  • Project URL: Click here to view the report.

The goal was to create an a-arm using numerical simulation concepts. Using SOLIDWORKS, I created design loading studies by simulating the vehicle travelling over a speed bump. From there I could analyze the stress and strain on my A-arm through mesh convergence and optimize it in each study. Lastly, I did performing and manufacturing and cost analysis to see how much the design would cost.
