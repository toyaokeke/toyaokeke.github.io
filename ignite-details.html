Online Learning Platform

Project Information

  • Category: Full Stack Development
  • Client: University of Calgary
  • Company: Aranite Inc.
  • Project URL: Schulich Ignite Official Website and the Ignite Platform

This is a full stack application to help high school students learn programming. It provides video streaming, chat rooms, coding environments, and tutorials / file sharing from TAs.

The platform eliminates having to set up your own development environment. To access it, go to the Ignite Platform and login using Gmail. Navigate to File>New>Notebook at the top left corner and then select Python 3 to begin coding.
